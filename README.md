# 🩸 LifeLink AI

🚑 AI-Powered Blood Resource Optimization System

## 🔍 Problem

Traditional blood search methods rely on WhatsApp and contacts, causing delays in emergencies.

## 💡 Solution

LifeLink AI intelligently recommends the best blood source (donor or blood bank) using:

* Machine Learning
* Geospatial Analysis
* Optimization Algorithms

## ⚙️ Features

* 🧠 AI-based donor ranking
* 📍 Location-based search
* 🏥 Blood bank availability
* 🚨 Smart decision system

## 🛠️ Tech Stack

* Python
* Streamlit
* Scikit-learn
* NumPy
